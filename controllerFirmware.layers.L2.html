controllerFirmware.layers.L2.custom_formatter module¶

This module is responsible for formatting data and command values. Data gets converted from raw value(hardware) to formatted value(acceptable in DeJoule). Command gets converted from formatted value to raw value. It uses unit module to convert raw units to DAU ( DeJoule Acceptable Units ) and vice-versa.

convert(value, invertConversion=False)¶

This is the main unit converter method which converts units as well as operates int/float on the converted values if needed.

Parameters

  • value (int/float) – the value which is to be converted.

  • invertConversion (bool) – if False conversion is from raw unit to dau,if True conversion is from dau to raw unit

Returns

converted and special operated value.

Return type

float/int

Raises

  • UnitConversionError – Error while converting units.

  • UnitNameError – Error when unit is not present in ureg units store.
